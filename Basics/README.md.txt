# Basics C Programs

This folder contains beginner C programs from the first semester:

- 'tourism-guide.c' → displays countries, cities & famous places
- 'fsc-dms.c' → calculates FSC marks sheet
- 'electricity-bill.c' → computes electricity bill based on slabs
- 'leap-year.c' → checks leap year
- 'bmi-calculator.c' → calculates BMI
- 'bank-loan.c' → checks loan eligibility & interest
- 'income-tax.c' → calculates income tax
- 'hr-screening.c' → screens job applicants
- 'helloworld.c' → prints "Hello, World!" on the screen
