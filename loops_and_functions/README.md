The repository includes the following C programs:

 -check-prime – Check whether a number is prime or not
 -count-digits – Count the number of digits in a given number
 -decimal-to-binary – Convert a decimal number into binary
 -fibonacci-series – Print Fibonacci series up to N terms
 -gcd-hcf – Find GCD/HCF of two numbers
 -power-of-num – Calculate the power of a number
 -print-even – Print even numbers up to N
 -print-odd – Print odd numbers up to N
 -print-factorial – Find factorial of a number
 -print-pyramid – Print a pyramid pattern using stars
 -reverse-a-num – Reverse a given number
 -sum-of-digits-of-num – Calculate sum of digits of a number
 -sum-of-N-even-num – Find sum of first N even numbers
 -sum-of-N-odd-num – Find sum of first N odd numbers
 -sum-of-N-num - Find sum of N natural numbers
